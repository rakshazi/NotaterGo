NotaterGo

Simple notes app (GTK2) with managing from system tray

View the Project on GitHub rakshazi/NotaterGo

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Features

  • Create note from tray
  • Autosave any note
  • All notes in plain text format That's all. Just create/read/delete notes, no more.

Folders

NotaterGo releases FreeDesktop specification for data dir.

Default: $XDA_DATA_HOME/NotaterGo

If $XDA_DATA_HOME env not set, will be used $HOME/.local/share/NotaterGo dir.
